# AKPay Mobile Wallet Application

AKPay is a cross-platform mobile wallet application developed using **Flutter** for both **iOS** and **Android** platforms.

The project demonstrates mobile development concepts including authentication, API integration, offline storage, push notifications, testing, and UI/UX design.

---

## Features

* User Authentication
* Wallet Dashboard
* Transaction History
* Send Money Functionality
* API Integration
* Offline Capability
* Push Notifications
* Testing & Optimization

---

## Technologies Used

* Flutter
* Dart
* REST API Integration
* SharedPreferences
* Firebase Cloud Messaging
* Git & GitHub

## Running the Application

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

Run tests:

```bash
flutter test
```

## Objective

The goal of AKPay is to demonstrate cross-platform mobile development using Flutter while implementing practical mobile application features and software engineering practices.
