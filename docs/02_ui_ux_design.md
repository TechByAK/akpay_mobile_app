# AKPay UI/UX Design

## Design Overview

The AKPay application is designed to be simple, modern, and easy to use. Since the app is related to mobile payments, the interface focuses on clarity, security, and fast navigation.

The design will support both iOS and Android using Flutter.

---

## Main Screens

### 1. Splash Screen

The splash screen displays the AKPay logo when the app starts.

### 2. Login Screen

The login screen allows users to enter their email and password to access their account.

### 3. Register Screen

The register screen allows new users to create an AKPay account.

### 4. Home / Wallet Dashboard

The home screen displays:

* user name
* wallet balance
* quick actions
* recent transactions

### 5. Send Money Screen

This screen allows users to enter a receiver, amount, and payment note.

### 6. Transaction History Screen

This screen displays a list of previous transactions.

### 7. Profile Screen

The profile screen allows users to view account details and log out.

---

## User Flow

1. User opens the app.
2. Splash screen appears.
3. User logs in or registers.
4. User reaches the wallet dashboard.
5. User can send money, view transactions, or access profile.
6. User logs out when finished.

---

## UI Style

The AKPay interface will use:

* clean layout
* simple buttons
* readable typography
* card-based wallet display
* clear icons
* professional fintech-style colors
