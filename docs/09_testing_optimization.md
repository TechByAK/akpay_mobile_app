# Testing and Optimization

## Overview

Testing and optimization are important steps in mobile development. They ensure that AKPay works correctly, performs well, and provides a smooth user experience.

## Testing Types

### Unit Testing

Unit tests check individual functions or classes.

### Widget Testing

Widget tests check Flutter UI components.

### Manual Testing

Manual testing verifies the app flow on iOS and Android devices.

## Tested Features

The following features were tested:

- login form validation
- register form validation
- wallet dashboard display
- transaction list display
- navigation between screens
- logout functionality

## Optimization

The app was optimized by:

- keeping UI simple and lightweight
- organizing code into folders
- using reusable services
- reducing unnecessary complexity
- preparing local storage for offline access

## Conclusion

Testing and optimization helped improve the reliability, usability, and maintainability of the AKPay mobile application.
