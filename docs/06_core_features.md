# Core Features

## Overview

The AKPay application includes core mobile wallet features that allow users to manage their wallet and transactions.

## Main Features

### 1. Wallet Balance

Users can view their current wallet balance on the home dashboard.

### 2. Transaction History

Users can view previous payments and received transfers.

### 3. Send Money

Users can enter a receiver and amount to simulate a money transfer.

### 4. Profile

Users can view basic account information and log out.

## Conclusion

The core features provide the main functionality expected from a mobile wallet application.
